# Security Policy

This repository is a sanitized public mirror of DeployKube.

## Reporting a vulnerability

Please do not open a public issue with exploit details, credentials, or environment-specific attack guidance.

Instead, report security issues privately to the maintainer through a direct channel.

## Public mirror scope

This public mirror is a sanitized publication of the codebase. Sensitive deployment details, credentials, and internal operational material were removed or replaced before publication.

The absence of a detail in this mirror does not imply the private working repo lacks it; some material is intentionally withheld for operational safety.
