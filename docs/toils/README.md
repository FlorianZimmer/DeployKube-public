# Toils

Toils are **operational how-tos** and troubleshooting notes that are not directly tied to a specific alert/incident.

Rules of thumb:
- If a Prometheus/Mimir alert links to a doc via `runbook_url`, that doc belongs in `docs/runbooks/`.
- If it’s a planned end-to-end playbook (bootstrap/release/restore), it likely belongs in `docs/guides/`.
