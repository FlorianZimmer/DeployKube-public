- _2026-03-10_ – Fixed a render-time RBAC regression that broke `cert-smoke-gateway-sni` on Proxmox: the base `kustomization.yaml` had a global `namespace: cert-manager` transformer, which rewrote the intended `Role/RoleBinding cert-smoke-istio-system` into `cert-manager`, so the smoke ServiceAccount could not read `Gateway/public-gateway` in `istio-system`. The fix removed the global namespace transformer, kept explicit namespaces on all namespaced resources, and extended `./tests/scripts/validate-certificates-smoke-tests-contract.sh` to assert that the proxmox render preserves `cert-smoke-istio-system` in `istio-system`.
