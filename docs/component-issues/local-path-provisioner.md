(ids: `dk.ca.finding.v1:local-path-provisioner:f6384d91a6a92316b029a8fcb10d5714b31aaa771209467cb5b067c46a5a8842`, `dk.ca.finding.v1:local-path-provisioner:f000fc3fc3e9a25755c20a26591ba30df54517bc9b5ae60481d0425bd3b7d9e9`, `dk.ca.finding.v1:local-path-provisioner:0c9ab414e5ce049e27f19db08c67c9917bd46a1bf4f52678c2c3c74eba73801b`)

#### capacity-and-performance-hygiene
- **`local-path-provisioner` has no CPU/memory requests/limits**: set realistic requests/limits to make scheduling/eviction predictable and reduce susceptibility to resource exhaustion; pair with alerting so provisioning degradation is detected quickly. (ids: `dk.ca.finding.v1:local-path-provisioner:6f965e03fc0e837b9ac31018e865120d4d437debf55acc46fe1cbadff3d71579`, `dk.ca.finding.v1:local-path-provisioner:d26e496cfe3df8f7471f439b97a24ca8bd548f89998db629a7f84db661abbfb4`, `dk.ca.finding.v1:local-path-provisioner:ec7e9d00af635113369bc3a81b994d5cf76aa59a58f0b75b5474cc6c5f55a1e0`)

#### data-protection-at-rest
- **No encryption-at-rest boundary is evidenced for node-local hostPath volumes**: document the supported data-classification boundary for `shared-rwo`; for sensitive workloads require an encrypted backend or verified node disk/filesystem encryption with clear key management. (ids: `dk.ca.finding.v1:local-path-provisioner:28447476576ce6f95533455184e84c81c0b2f1d9e74914cfa145e2e9a628de85`)

- **Provisioner/helper pod security hardening is not declared (incl. non-root where possible)**: set explicit pod/container `securityContext` defaults (seccomp, drop caps, disallow privilege escalation, readOnlyRootFilesystem where feasible, runAsNonRoot/UID/GID where supported) and document any required root exceptions for hostPath operations. #### high-availability
- **Node-affined hostPath storage has no failover (node/disk loss = outage/DR event)**: keep `shared-rwo` scoped to single-node/platform-owned use; migrate HA/multi-node workloads to replicated storage. (ids: `dk.ca.finding.v1:local-path-provisioner:aa2f21670f3436672bbdb6ad430f1fcfc0b7513adca4939d8dfed1123a52e81b`)

(ids: `dk.ca.finding.v1:local-path-provisioner:11e7c292447f19c3355f1fbbdc77e34d63c22a949f01d55714928591cf89f19c`, `dk.ca.finding.v1:local-path-provisioner:edf59c36645915d73c38828b67f12fa943416fb68ed1edbe3e7601430788962c`)
