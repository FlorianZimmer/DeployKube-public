### High

#### anti-affinity
- Make replica placement failure-domain-safe: today placement relies on *preferred* anti-affinity and lacks explicit spreading; combined with `externalTrafficPolicy: Local` this can become a total DNS outage if both replicas land on one node. Adopt `requiredDuringSchedulingIgnoredDuringExecution` where feasible and/or add `topologySpreadConstraints`.
